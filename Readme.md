# BAMBAM

BAMBAM is a Windows-based application designed for Packing. The application features a user-friendly interface that allows users to Pack binaries

## Features

- **Compression Tab**: Compress files and folders using various algorithms.
- **Options Tab**: Customize application settings according to user preferences.
- **Log Tab**: View logs and debug information for troubleshooting.
- **About Dialog**: Information about the application and its developers.

## Requirements

- Windows 
- Microsoft Visual C++ Redistributable 

## Usage

1. **Launch the Application**: Double-click on `bambam.exe`.
2. **Navigate Tabs**: Use the tab control at the top to switch between Compression, Options, and Logs.
3. **Compress Files**: 
   - Go to the Compression tab.
   - Select files or folders to compress.
   - Choose your desired compression settings and click "Compress".
4. **Adjust Settings**: 
   - In the Options tab, configure your preferences.
5. **View Logs**: 
   - Check the Logs tab for real-time updates and debug information.
